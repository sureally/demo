# 几个有趣的操作

## 转为小写
```
('a' | ' ') = 'a'
('A' | ' ') = 'a'
```

## 转为大写
```
('b' & '_') = 'B'
('B' & '_') = 'B'
```

## 大小写互换
```
('d' ^ ' ') = 'D'
('D' ^ ' ') = 'd'
```

## 判断两个数字是否异号
```
int x = -1, y = 2;
bool f = ((x ^ y) < 0); // true

int x = 3, y = 2;
bool f = ((x ^ y) < 0); // false
```

## 消除数字 n 的二进制表示中的最后一个 1
```
n & (n-1)
```
